### 1. React Hook Form

\: React에서 Form을 쉽게 만들기 위한 라이브러리<br/> 성능이 좋고 유연하며 유효성 검사에 아주 탁월함

- 특징

  - 비제어 컴포넌트 방식(사용자가 타이핑하거나 값을 변경할 때 리랜더링을 줄여줌)
  - 데이터는 계속 동기화되며 분산되지 않아 데이터를 한 곳에서 관리하고자 하는 니즈도 충족 가능

- **장점**
  - 적은 코드로 더 좋은 퍼포먼스
  - 다른 라이브러리나 React에 비해 Rerender 수가 적음
  - Fast Mounting(로딩속도가 빠름)
  - TS를 기본으로 지원

### 2. React Hook Form 사용법

```bash
npm install react-hook-form
yarn add react-hook-form
```
