# 푸드파이트 대회 Algorithm(Lv.1)

<br/>

문제 설명

-    정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

<br/>

제한사항

-    n은 0 이상 3000이하인 정수입니다.

<br/>

입출력 예시
<br/>

| n   | return |
| --- | ------ |
| 12  | 28     |
| 5   | 6      |

<br/>

```javascript
function solution(n) {
     let answer = 0;
     for (i = 1; i < n + 1; i++) {
          //   if (n % i == 0) {
          //        answer += i;
          //   } 아래의 삼항연산자와 같은 내용
          n % i == 0 ? (answer += i) : null;
     }
     return answer;
}
```

<br/>

<br/>

### 나름 풀이

for 반복문으로 1부터 n번째까지 i%n = 0(즉 n의 약수 구하기)인 값들을 삼항연산자(if문)로 골라내고 -> 해당 값들의 합을 구한다.
