# [1차] 비밀지도(Lv.1)

<br/>

문제 설명

- 지도는 한 변의 길이가 n인 정사각형 배열 형태로, 각 칸은 "공백"(" ") 또는 "벽"("#") 두 종류
- 전체 지도는 두 장의 지도를 겹쳐서 얻을 수 있음(각각 "지도 1"과 "지도 2"라고 하자. 지도 1 또는 지도 2 중 어느 하나라도 벽인 부분은 전체 지도에서도 벽이다. 지도 1과 지도 2에서 모두 공백인 부분은 전체 지도에서도 공백)
- "지도 1"과 "지도 2"는 각각 정수 배열로 암호화
- 암호화된 배열은 지도의 각 가로줄에서 벽 부분을 1, 공백 부분을 0으로 부호화했을 때 얻어지는 이진수에 해당하는 값의 배열

<img src="http://t1.kakaocdn.net/welcome2018/secret8.png" alt="programmers" width="600"/>

<br/>

입력 형식

- 입력으로 지도의 한 변 크기 n 과 2개의 정수 배열 arr1, arr2가 들어옴
- 1 ≦ n ≦ 16
- arr1, arr2는 길이 n인 정수 배열
- 정수 배열의 각 원소 x를 이진수로 변환했을 때의 길이는 n 이하이다. 즉, 0 ≦ x ≦ 2n - 1을 만족

<br/>

입출력 예시
<br/>

| 매개변수 | 값                                          |
| -------- | ------------------------------------------- |
| n        | 3                                           |
| arr1     | [9, 20, 28, 18, 11]                         |
| arr2     | [30, 1, 21, 17, 28]                         |
| 출력     | ["#####","# # #", "### #", "# ##", "#####"] |

<br/>

```javascript
function solution(n, arr1, arr2) {
  var answer = [];
  for (i = 0; i < n; i++) {
    let x = (arr1[i] | arr2[i]).toString(2);
    answer.push(x.padStart(n, "0").replace(/1/g, "#").replace(/0/g, " "));
  }
  return answer;
}
```

#### 시작 전에 새로운 메소드들 정리

- toString() : 문자열을 반환하는 대표적인 메소드이지만, 여기서는 2진수로 변경하기 위해 사용했음
  - toString(2) 의 경우 2진수로 변환함
- padStart() : 현재 문자열의 시작을 다른 문자열로 채워, 주어진 길이를 만족하는 새로운 문자열을 반환함
  - 끝에서부터 시작할 땐 padEnd()
- \| : 비트 연산자(비트 OR 연산) → 대응되는 비트 중에서 하나라도 1이면 1을 반환
  - 비트단위로 논리 연산을 수행함<br/>

#### 풀이 시작

1. 먼저 둘 중 하나라도 1이 존재하면 채워진 값으로 들어가야함 → 비트 OR 연산을 통해 설정
2. 배열의 길이는 n이므로 비어있는 값은 0으로 채워줌(padStart)
3. 1은 #으로, 0은 공백(" ")으로 넣어줌
4. 완성!
