## 1. Typescript란

- 사용하는 이유
- 특징
  - strongly typed 프로그래밍 언어
  - 타입스크립트로 작성한 코드를 컴파일하면 자바스크립트 코드가 됨 but, 타입스크립트 상에서 에러가 있으면 자바스크립트로 컴파일되지 않음

### 2. Type의 접근방식

1. 명시적 정의(변수 선언 시 타입 정의)
2. 변수만 생성(타입 추론)
   /: 명시해주지 않아도 TS 내에서 알아서 타입을 추론해줌

```typescript
// 1. 명시적 정의
let 현징: boolean = true;

// 2. 타입 추론
let 현징 = true;

// 위의 두가지는 같은 타입을 가짐
```
