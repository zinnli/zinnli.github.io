## - Axios -

<br/>

### **Axios**

: node.js와 브라우저를 위한 Promis 기반의 HTTP 클라이언트
<br/>

#### 설치방법

```react
//yarn
$ yarn add axios

//npm
$ npm install axios
```

<br/>

#### 특징

- 브라우저를 위해 XMLHttpRequests(XHR) 생성 : 서버와 상호작용할 때 사용
- node.js를 위해 http 요청 생성
- Promise API를 지원
- 요청 및 응답 인터셉트
- 요청 및 응답 데이터 변환
- 요청 취소
- JSON 데이터 자동 변환
- XSRF를 막기위한 클라이언트 사이드 지원
