- **사이드 프로젝트 지역 선택 건으로 논의진행**

  - 원래 진행하려고 했던 방법 : 지역 리스트 DB가 들어있는 오픈 API를 찾아서 사용하려고 했음<br/>
    → 이틀가량 해당 API를 찾았으나 찾지 못해 구글링 및 주위의 개발자분께 여쭤본 결과

    1. 백엔드에서 csv 파일을 받아서 디비화 시키고 쿼리 날려서 사용(백엔드 개발자님 조언 ← DB 쓸 수 없는 상황엔 JSON 쓴다고 함)
    2. 오픈 API 더 검색해서 DB로 설정할 수 있는지 확인해보고 진행하기
    3. 카카오맵 API 사용하여 주소를 직접 검색하고 원하는 시군구만 따와서 백에 넘기기

  - 위처럼 3가지 안이 생겼고, 같이 사이드 프로젝트 진행하는 프엔분과 논의해봤는데 결과는 "오픈 api를 사용하는게 좋을 것 같다(내가 하고싶다고 함) + 하고싶은대로 해도 좋을듯" 이었다.
  - 현재 공공데이터 포탈에 있는 시군구 데이터가 있어 인증키 발급 요청해둔 상태라 결과 받는대로 진행해야할듯.<br/><br/>

- react\-router\-dom의 outlet 공부해봐야할듯
