3. 그리고 항상 헷갈리는 mac에서 git push 하는 순서 ⭐️⭐️⭐️⭐️⭐️
   - 큰 순서는 기억하자!
     1. git init(git에 레포 연결 안해놨을 때1)
     2. git add . (전체 업로드 - .gitignore로 올라가면 안되는 파일들 추려두기)
     3. git commit -m "message"
     4. git remote add origin "github.com/repo.git" (git에 레포 연결 안해놨을 때2)
     5. git push -u origin master(git에 레포 연결 안해놨을 때3)
     6. (git에 올리기만 하면 되는 경우) git push
