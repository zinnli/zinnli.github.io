<br/>
이번주는 멘토님에게 피드백 받는 시간도 가졌다.
<br/>
생각보다 추가할 부분이 많았고 일주일동안 작업한 부분들이 리셋된 느낌이 컸다. 그래도 덕분에 조금 정리된 느낌이 있었지만 스코프가 생각보다 조금 더 커진 느낌이라 많이 걱정이 되었다.
<br/>
<br/>

내게 할당된 페이지들이 생각보다 기능들의 구현이 조금 복잡한 편이었고, 와이어 프레임 자체도 생각보다 늦게 나오다보니 뭔가 나도 모르게 조금 마음을 가볍게 먹었던 것 같다. 추가로 백엔드 팀에서 해킹 이슈로 배포가 다시 롤백되다보니 이것저것 핑계삼아 일주일을 날린 것 같은데 다시 마음가짐을 제대로 먹는게 중요할 것 같다.
<br/>
<br/>

검색하는 부분이 제일 문제일 것 같아 미리 구글링을 많이 해뒀긴 한데 뭔가 서버 연결 안된 상태로 뭔가 진행한다는게 쉽지않은 것 같다.
<br/>
<br/>

다른 팀 보니 혼자서도 포스트맨 같은 부분을 활용하여 테스트하던데 항해가 끝나고 나면 해당 부분도 한 번 고민해보는 게 좋을 것 같다.
<br/>
